## LaunchpadPro 1.1.0

LaunchpadPro is a native macOS app launcher with a full-screen grid, folders, search, trackpad paging, and saved layout memories.

### Highlights

- Universal macOS build for Apple Silicon and Intel Macs
- Full-screen paged app grid
- Fast search
- Drag-and-drop app ordering
- Folder creation, renaming, scrolling, and drag-out support
- Up to 3 saved layout memories
- Menu bar icon visibility setting
- Hotkey and hot corner launch options
- Local-only layout persistence

### Install

Download `LaunchpadPro.dmg`, open it, and drag `LaunchpadPro.app` into `Applications`.
